## Detailed Description
```
Stop accidentally sharing sensitive data with AI chatbots.

Prism AI is a lightweight privacy shield that monitors what you type in ChatGPT, Claude, and Gemini. It instantly detects sensitive information and helps you redact it before hitting send.

WHAT IT DETECTS:

Critical (Red alerts):
- API keys & tokens (AWS, GitHub, Slack, etc.)
- Passwords & secrets
- Credit card numbers
- Social Security Numbers
- Private keys & JWTs

Sensitive (Amber alerts):
- Email addresses
- Phone numbers
- Physical addresses
- Full names
- Dates of birth

Contextual (Blue alerts):
- IP addresses
- Internal URLs
- Company names
- Project codenames

KEY FEATURES:

Real-time Detection
The minimal pill indicator shows your privacy score (0-100) as you type. Red border = danger, green = safe.

One-Click Redaction
Click "Redact All" to instantly replace sensitive data with safe placeholders like [EMAIL], [PHONE], [API_KEY].

Risk Explanations
Click on any detected item to understand WHY it's flagged and what the risks are.

Ghost Mode
Blur all input fields when someone walks by. Perfect for working in public.

Auto Dark/Light Mode
Automatically matches your AI chat's theme.

Keyboard Shortcuts
- Cmd/Ctrl + Shift + P → Redact All (Protect)
- Cmd/Ctrl + Shift + G → Toggle Ghost Mode

100% PRIVATE:

- Runs entirely on your device
- Zero data collection
- No external servers
- No analytics or tracking
- Open source - audit the code yourself

Works on:
- ChatGPT (chat.openai.com, chatgpt.com)
- Claude (claude.ai)
- Google Gemini (gemini.google.com)

Protect yourself from accidental data leaks. Install Prism AI today.
```
